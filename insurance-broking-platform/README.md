# AtriSure Nexus · Insurance Broking OS

AtriSure Nexus is a no-code, AI-enriched operating system crafted for insurance intermediaries. Eleven interconnected modules converge into a single workspace that orchestrates client relationships, policy workflows, claims, compliance, finance, analytics, partner experiences, and institutional knowledge—without writing code.

## Brand System

- **Platform Name:** AtriSure Nexus  
- **Descriptor:** Insurance Broking Operating System  
- **Palette:**  
  - **AtriSure Blue** `#2C5FF6` – hero tones, call-to-action elements  
  - **Neon Aqueous** `#1AC6D1` – AI indicators, highlights  
  - **Solar Amber** `#FFB454` – escalations, reminders, proactive nudges

## Eleven Connected Modules

1. **Client Orbit** – 360° client intelligence and relationship mapping  
2. **Policy Forge** – drag-and-drop policy assembly with compliance guardrails  
3. **Claims Prism** – proactive triage, anomaly detection, and orchestration  
4. **Carrier Nexus** – API exchange, shared deal rooms, and collaboration  
5. **Quote Lab** – scenario-driven quoting and market benchmarking  
6. **Billing Vault** – cash application radar and commission intelligence  
7. **Compliance Archive** – license guardian and audit-ready evidence lockers  
8. **Analytics Pulse** – predictive KPI cloud and narrative insights generator  
9. **Workflow Studio** – automation canvas and playbook marketplace  
10. **Partner Bridge** – co-branded partner portals and co-broking workbench  
11. **Knowledge Companion** – AI concierge and living playbooks

## Universal Feature Layer

- Visual **No-Code Studio** with drag-and-drop canvas, widget inspector, and data binding placeholders  
- **Smart Intake Console** featuring CRUD operations, AI autocomplete, duplicate detection, and merge controls  
- **Bulk Import/Export** via CSV with auto mapping, dataset summaries, and download utilities  
- **Document Intelligence** simulator for OCR-like policy parsing  
- **Natural Language Search** across all modules and features  
- **AI Concierge Chatbot** for guided assistance and instant module navigation

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000 to explore the platform.

## Production Build

```bash
npm run build
npm start
```

AtriSure Nexus is built with Next.js App Router, TypeScript, Tailwind CSS v4 custom tokens, lucide-react icons, PapaParse utilities, and lightweight client-side AI simulators (no external AI APIs required).
